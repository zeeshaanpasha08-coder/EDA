This project performs simple EDA on the Titanic dataset.
The goal is to understand the data using statistics and visual charts.

* What I did

Loaded the Titanic dataset
Checked basic info and missing values
Created summary statistics
Plotted histograms
Plotted boxplots
Created a correlation heatmap
Used pairplot to see relationships

* Tools Used

Python
Pandas
Seaborn
Matplotlib
